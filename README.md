# Atomic Dota

**Atomic Dota** is a research-oriented software engineering and digital platform design project focused on safe online interaction, habit-oriented engagement, and community moderation.

This repository accompanies the academic report:
**"Atomic Dota: Designing a Safe, Habit-Oriented Social Platform with an AI-Assisted Moderation Layer"**

## Overview
Atomic Dota was originally developed as an undergraduate software engineering project and later expanded into a full academic report for postgraduate application purposes. The project explores how software architecture, moderation design, user-centred workflows, and prototype engineering can be combined to support safer digital communities.

## Research Focus
The project investigates the following question:

> How can a prototype social platform be designed to support user expression, habit-oriented engagement, and safer online interaction through structured moderation and system design?

## Main Components
- User registration and login
- User profile and post interaction
- Text-based communication module
- AI-assisted moderation concept
- Admin management module
- Privacy and safety-oriented platform rules
- Software engineering artefacts such as SRS, DFDs, testing logic, and effort estimation

# Methodology

The project follows a software-engineering-oriented research methodology that combines:

1. Problem framing based on unsafe or low-quality online interaction.
2. Functional decomposition of core platform modules.
3. Prototype architecture design.
4. Risk analysis and requirement specification.
5. Conceptual AI-assisted moderation logic.
6. Evaluation through software engineering artefacts, testing logic, and effort estimation.

This methodology is suitable for an academic project report where the goal is to show structured technical reasoning and independent analytical writing.
